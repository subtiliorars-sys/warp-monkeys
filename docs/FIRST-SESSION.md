# First session — honesty brief

**Audience:** new players / playtesters  
**Purpose:** set expectations for the first Warp Division hop

## What this game is

- Meta fleet comedy: **Warp Monkeys** (+ Space Dandies crossover fuel) on Phaser 3 + TypeScript.
- Original characters — CodeMonkeys platform comedy offshoot, **not** TV IP.
- Local play + optional GitHub Pages / itch distribution.

## What you will **not** see

- FOMO timers or streak-shame for leaving mid-hop
- Pay-to-progress / loot-box framing in the core loop
- Forced account for local `npm run dev`

## Suggested first minutes

1. `npm install` → `npm run dev`.
2. Hop once; note coin/fuel clarity.
3. `npm run verify` before any PR.
4. Feature open: [PR #5](https://github.com/subtiliorars-sys/warp-monkeys/pull/5) (Time Echo + crossover fuel) — leave merge to Mark.
5. Do **not** run `npm run push:itch` without Mark (needs butler secrets).
